## 🏗️ Kiến Trúc Mã Nguồn (Static Site Structure)
Dự án được xây dựng theo kiểu CMS tĩnh để tối ưu hóa tốc độ tải trang và SEO:

```text
blog-site/
├── index.html              # Trang chủ (Hiển thị danh sách bài viết mới nhất)
├── sitemap.xml             # Sơ đồ trang web hỗ trợ Google Index
├── robots.txt              # Chỉ dẫn cho Bot tìm kiếm
│
├── pages/                  # CHUYÊN MỤC & TRANG TĨNH
│   ├── basketball/         # Chuyên mục: Bóng rổ
│   │   └── index.html      # (URL: /pages/basketball/)
│   ├── e-sports/           # Chuyên mục: Thể thao điện tử
│   │   └── index.html      # (URL: /pages/e-sports/)
│   ├── football/           # Chuyên mục: Bóng đá
│   │   └── index.html      # (URL: /pages/football/)
│   └── volleyball/         # Chuyên mục: Bóng chuyền
│       └── index.html      # (URL: /pages/volleyball/)
│
├── posts/                  # CHI TIẾT BÀI VIẾT (Cấu trúc Folder-based)
│   ├── 20-04-2026-post-1/  # Thư mục bài viết cụ thể
│   │   └── index.html      # Nội dung chi tiết bài viết 1
│   └── 20-04-2026-post-2/
│       └── index.html      # Nội dung chi tiết bài viết 2
│
├── components/             # CÁC THÀNH PHẦN TÁI SỬ DỤNG (Modular)
│   ├── header.html         # Thanh điều hướng dùng chung
│   └── footer.html         # Chân trang dùng chung
│
├── assets/                 # TÀI NGUYÊN HỆ THỐNG
│   ├── css/                # Tệp tin style.css
│   ├── js/                 # main.js (Xử lý giao diện) & router.js (Điều hướng)
│   └── img/                # Kho lưu trữ hình ảnh bài viết và giao diện
│
└── data/                   # DỮ LIỆU HỆ THỐNG
    └── posts.json          # "Trái tim" dự án: Chứa thông tin bài viết để Render tự động
```
