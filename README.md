## 🏗️ Kiến Trúc Mã Nguồn (Static Site Structure)
Dự án được xây dựng theo kiểu CMS tĩnh để tối ưu hóa tốc độ tải trang và SEO:

```text
blog-site/
│
├── index.html              # Trang chủ
├── sitemap.xml             # Sơ đồ trang web
├── robots.txt              # Cấu hình bot tìm kiếm
│
├── pages/                  # Các trang tĩnh
│   ├── about.html          # Giới thiệu
│   └── contact.html        # Liên hệ
│
├── posts/                  # Thư mục bài viết
│   ├── post-1/
│   │   └── index.html
│   ├── post-2/
│   │   └── index.html
│
├── assets/                 # Tài nguyên hệ thống
│   ├── css/                # style.css
│   ├── js/                 # main.js & router.js
│   └── img/                # Hình ảnh
│
├── components/             # Thành phần tái sử dụng
│   ├── header.html
│   └── footer.html
│
└── data/
    └── posts.json          # Danh sách dữ liệu bài viết
```
